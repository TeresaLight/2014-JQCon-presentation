Overview of CreateJS

CreateJS is a suite of modular libraries and tools which work together to enable rich interactive content on open web technologies via HTML5. EaselJS

A Javascript library that makes working with the HTML5 Canvas element easy. About EaselJS. EaselJS provides straight forward solutions for working with rich graphics and interactivity with HTML5 Canvas.

Easy to use flash-like API

Animate Sprites

Filters, Masks and Transformations

Example Display Object using Create JS

								

								var g = new createjs.Graphics();
								g.setStrokeStyle(1);
							    
							    g.beginStroke(createjs.Graphics.getRGB(0,0,0));
							    g.beginFill(createjs.Graphics.getRGB(255,0,0));
							    g.drawCircle(0,0,3);

PreloadJS

A Javascript library that lets you manage and co-ordinate the loading of assets

Control page load speeds by chosing when to load what

TweenJS

A Javascript library for tweening and animating HTML5 and Javascript properties.

Simple API that allows for chaining.
